# Eenvoudige Contactenlijst Applicatie

Ontwikkel een eenvoudige contactenlijst applicatie in C# met behulp van het .NET Framework. De applicatie moet de volgende functionaliteiten bieden:

1.  **Contact Toevoegen:** Gebruikers moeten nieuwe contacten kunnen toevoegen met een naam en telefoonnummer.
2.  **Contacten Weergeven:** Alle toegevoegde contacten moeten worden weergegeven.
3.  **Contact Zoeken:** Gebruikers moeten kunnen zoeken naar contacten op naam.

Dit project is bedoeld om uw basiskennis van C# en het .NET Framework te testen, inclusief het werken met klassen, objecten en eenvoudige invoer/uitvoer.

## Opdracht

Implementeer de gevraagde functionaliteit.
